DayZ Monitoring Bot (ранняя версия)

Компактный бот для Discord, который отображает онлайн, игровое время и очередь на DayZ сервере прямо в статусе бота.

Для запуска:

Установите библиотеки: discord.py и aiohttp

В файле bot.py (строка 328) укажите токен бота

В файле config.json настройте IP, Query порт, токен и шаблон отображения

⚠️ Версия в разработке — скоро будет полностью автоматический установщик без необходимости вручную устанавливать зависимости. Следите за обновлениями!
-------------------------------------------------------------------------------------------------------------------------------------------------------
Установите библиотеки:
{ pip install discord.py aiohttp ]

Обновите pip (рекомендуется)
{ python -m pip install --upgrade pip }

Нужные библиотеки: 
**discord.py**
**aiohttp**
